# Changelog

## [0.1.0] - 2026-02-12

### Added

- Tree-sitter-based diagnostics (syntax errors and arity validation)
- Completion provider for MeTTa builtins and user-defined symbols
- Hover provider with documentation for all builtins
- Go-to-definition for user-defined functions and variables
- Document symbols and workspace symbol search
- Code folding for multi-line S-expressions
- Code actions with quick-fixes for mismatched brackets
- Restart Language Server command
